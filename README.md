# http://wxkb.io/
Weather forecast in under 10kb.



# Run app with Node
1. clone repo
2. `npm install`
3. `FORECAST_IO_API_KEY=<api key goes here> node_modules/nodemon/bin/nodemon.js`


### Compile CSS
* Automatically compile: `npm run watch-css`
* If you just want to run it once: `npm run build-css`
